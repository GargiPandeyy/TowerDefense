# Tower Defense Game

a simple tower defense game built with html, css, and javascript

## how to play

- place towers to stop enemies from reaching the end
- earn money by defeating enemies
- survive all 10 waves to win

## features

- 3 different tower types
- multiple enemy types
- wave-based gameplay
- tower upgrades
- simple and fun!

## how to run

just open index.html in your browser
